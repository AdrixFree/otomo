# otomo
Otomo - Free Radar + Assister for Interlude by LanGhost

Radar Mode:

E - Find next target
W - Change target class
Q - Change reskill range
B - Change target clan
A - Warlord ignore
S - Cancel
SPACE - Auto attack

Assist mode:

D - Move to assister
S - Cancel
SPACE - Auto attack
