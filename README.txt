# otomo
Otomo - Free Radar + Assister for Interlude by LanGhost

Radar Mode:

E - Find next target
W - Change target class
Q - Change reskill range
A - Warlord ignore
S - Cancel
R - Aura bolt + Aura Flash
SPACE - Auto attack

Assist mode:

D - Move to assister
S - Cancel
F - Aura bolt + Aura Flash
SPACE - Auto attack
